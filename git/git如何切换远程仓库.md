# git 切换远程仓库
[toc]
## 1.git remote -v
```
git remote -v
```
- 查看当前远程仓库
## 2.git remote set-url origin 切换的远程仓库链接
```
git remote set-url origin 想要切换的远程仓库链接
```
- 想要切换的远程仓库链接的格式
    - https格式
        - https://github.com/用户名/仓库名.git
    - ssh格式
        - git@github.com::用户名/仓库名.git
